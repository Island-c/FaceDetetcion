# About

本程序是一款windows软件，主要用了opencv3.1 + opencv_contrib + qt 5.6 + vs2015。

原先是做的人脸检测，后来加入了人脸识别和目标物体的追踪，最终对人脸打上马赛克。本人初学者，这些东西都是现学现卖，代码写得好烂。。
